# Performance Optimization Checklist

## ✅ Pre-Deployment Checklist

### Build & Bundle
- [ ] Run `npm run build:analyze`
- [ ] Verify total bundle < 1MB
- [ ] Verify initial JS < 500KB
- [ ] Check for code splitting
- [ ] No duplicate chunks
- [ ] All heavy libs lazy-loaded

### Images
- [ ] All images use OptimizedImage
- [ ] Hero images have priority=true
- [ ] Below-fold images lazy loaded
- [ ] Blur placeholders configured
- [ ] Fallback images set

### API & Data
- [ ] GET requests use cache
- [ ] Cache TTL configured
- [ ] Retry logic enabled
- [ ] Prefetching implemented
- [ ] No over-fetching

### React Performance
- [ ] Search inputs debounced
- [ ] Scroll handlers throttled
- [ ] Lists paginated/virtualized
- [ ] Heavy components memoized
- [ ] Stable callbacks used

### CSS
- [ ] Tailwind JIT enabled
- [ ] Unused styles purged
- [ ] Critical CSS inlined
- [ ] No unused imports

### Monitoring
- [ ] Web Vitals reporter active
- [ ] Datadog/Sentry configured
- [ ] Analytics endpoint working
- [ ] Performance warnings enabled

### Testing
- [ ] Lighthouse score > 90
- [ ] LCP < 2.5s
- [ ] FID < 100ms
- [ ] CLS < 0.1
- [ ] Test on slow 3G
- [ ] Test on low-end device

### Documentation
- [ ] Performance guide updated
- [ ] Examples documented
- [ ] Scripts documented
- [ ] Team trained

## ✅ Post-Deployment Checklist

### Monitoring
