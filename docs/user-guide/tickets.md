# Ticket Management Guide

Complete guide to creating and managing support tickets.

## Creating Tickets

### Basic Ticket Creation

1. Click **"Create Ticket"** or press `C`
2. Fill in required fields
3. Add attachments (max 10MB)
4. Click **"Submit"**

### Ticket Templates

Use templates for common issues:
- Hardware Request
- Password Reset
- Software Installation
- Bug Report

## Ticket Details

### Ticket Anatomy

- **ID**: Unique ticket identifier (e.g., #12345)
- **Title**: Brief description
- **Status**: Current state
- **Priority**: Urgency level (Low, Medium, High, Critical)
- **Category**: Issue type
- **Assigned To**: Agent handling the ticket
- **Created**: Timestamp
- **Last Updated**: Latest activity

### Priority Levels

- **Critical**: System down, blocking work
- **High**: Major issue affecting multiple users
- **Medium**: Issue affecting single user
- **Low**: Minor issue or question

## Updating Tickets

### Adding Comments

1. Open ticket
2. Scroll to "Add Comment"
3. Type your message
4. Click **"Add Comment"**

### Uploading Attachments

1. Click **"Attach File"**
2. Select file (max 10MB)
3. Click **"Upload"**

Supported formats: PDF, PNG, JPG, GIF, TXT, DOC, DOCX

## Tracking Progress

### Email Notifications

You'll receive emails when:
- Ticket is assigned
- Agent adds comment
- Status changes
- Ticket is resolved

### Real-time Updates

The ticket page updates automatically when changes occur.

## Closing Tickets

When your issue is resolved:

1. Verify the solution works
2. Add confirmation comment
3. Click **"Mark as Resolved"**

You can reopen tickets within 7 days if the issue persists.

## Tips for Effective Tickets

1. **Be specific**: Clear titles help agents route tickets
2. **Include details**: Version numbers, error messages, screenshots
3. **One issue per ticket**: Don't combine multiple problems
4. **Follow up**: Respond promptly to agent questions
5. **Confirm resolution**: Test the solution before closing

## Keyboard Shortcuts

- `C`: Create new ticket
- `S`: Search tickets
- `N`: Next ticket
- `P`: Previous ticket
- `/`: Focus search
