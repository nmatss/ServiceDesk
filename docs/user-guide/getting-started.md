# Getting Started with ServiceDesk

Welcome to ServiceDesk! This guide will help you get started quickly.

## First Login

1. **Access the application**: Navigate to your ServiceDesk URL
2. **Login credentials**: Use the credentials provided by your administrator
3. **First-time setup**: Complete your profile and enable 2FA (recommended)

## Dashboard Overview

After logging in, you'll see your personal dashboard with:

- **My Tickets**: All tickets assigned to or created by you
- **Recent Activity**: Latest updates on your tickets
- **Quick Actions**: Create ticket, search knowledge base
- **Notifications**: Real-time updates

## Creating Your First Ticket

1. Click **"Create Ticket"** button
2. Fill in the form:
   - **Title**: Brief description of the issue
   - **Category**: Select appropriate category
   - **Priority**: Select urgency level
   - **Description**: Detailed explanation
   - **Attachments**: Add screenshots if needed
3. Click **"Submit"**

Your ticket will be automatically assigned to the appropriate team.

## Navigating the Interface

### Main Menu

- **Dashboard**: Overview of your tickets and activity
- **My Tickets**: List of all your tickets
- **Create Ticket**: Submit new support request
- **Knowledge Base**: Search for solutions
- **Profile**: Manage your account settings

### Ticket States

- **New**: Just created, awaiting assignment
- **In Progress**: Being worked on by an agent
- **Waiting for User**: Agent needs more information from you
- **Resolved**: Issue fixed, awaiting your confirmation
- **Closed**: Ticket completed
