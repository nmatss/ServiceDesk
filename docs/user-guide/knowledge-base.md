# Knowledge Base Guide

Learn how to use the ServiceDesk knowledge base effectively.

## Searching Articles

### Quick Search

1. Enter keywords in search box
2. Press Enter
3. Browse results

### Advanced Search

Use filters to narrow results:
- **Category**: Filter by topic
- **Tags**: Find related articles
- **Date**: Recent or updated articles

### Search Tips

- Use specific keywords
- Try different terms if no results
- Check spelling
- Use quotes for exact phrases: "error 404"

## Article Structure

### Article Sections

Most articles include:
- **Summary**: Quick overview
- **Problem**: Issue description
- **Solution**: Step-by-step fix
- **Screenshots**: Visual guides
- **Related Articles**: Similar topics

### Article Ratings

Help improve articles:
- **Helpful**: Solution worked
- **Not Helpful**: Needs improvement
